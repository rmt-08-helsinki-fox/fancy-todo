# Fancy Todos


```Models: ```

**1. Todos table**
```
{
    "title": <string>,
    "description": <string>,
    "status": <string>,
    "due_date": <date>,
    "UserId": <integer>,
}
```
foreign key:
```  UserId  ```

validations:
* title: notEmpty
* description: notEmpty
* due_date: notEmpty, notPassedDate


**2. Users table**
```
{
    "email": <string>,
    "name": <string>,
    "password": <string>,
    
}
```

validations:
* email: isEmail, notEmpty
* name: notEmpty
* password: notEmpty, len[6]

``` API```

- https://newsapi.org/ => news api

## <a id="createTodos"></a>POST /todos/add
### *Create a todo object into database*
### Request Body
``` 
{
    "title": <string>,
    "description": <string>,
    "status": <string>,
    "due_date": <date>
    "UserId": <integer>
}    
```
### Request Headers
``` 
{
    "accessToken": <string>
}
```
### Responses:

* code 201: Successful operation. Return the added object
``` 
{
    "title": <string>,
    "description": <string>,
    "status": <string>,
    "due_date": <date>
}    
```
* code 401: error in authentication
```
{
    "error":
    {
        "code":401,
        "message":"jwt malformed"
    }
}
```
* code 400: Validation errors. 
```
{
    "error":
    {
        "code":400,
        "messages":
        [
        "Cannot input past day in due_date parameter","Validation notEmpty on title failed","Validation notEmpty on description failed"
        ]
    }
}
```
* code 500: internal server error
```
{
    "error": {
        "code": 500,
        "messages": "internal server"
    }
}
```
